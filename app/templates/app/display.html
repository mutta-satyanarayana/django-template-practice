Filter

  • upper Name using |: {{name | upper}}
  • lower Name using |: {{name|lower}}
  • lines break using | linesbreak : {{desc|truncatewords:10}}
  • characters break using | truncatechars:2: {{desc|truncatechars:2}}
  • truncate words & charcters: {{desc|truncatewords:10|truncatechars:10|upper}}
  • truncate words & charcters: {{desc|truncatewords:10|upper|truncatewords:10}}
  • capfirst using | {{desc|capfirst}}
  • title using | {{desc|title}}
  • default (if have value then show value otherwise show default value) | Ex-1: {{status|default:"No User"}} Ex-2: {{status1|default:"no data found"}}
  • length find for string or list : {{desc|length}} Ex-2(for status1 var): {{status1|length}} Ex-2(list length): {{list1|length}}
  • Slice using |: {{name|slice:5}}
  • Date using|: {{date|date:'D d m y'}}
  • Date Predefind Format : {{date|date:"SHORT_DATE_FORMAT"}} {{date|date:"DATE_FORMAT"}}
  • Time using|: {{date|time:'h:i:s a '}}
  • Time Predefind Format : {{date|time:"TIME_FORMAT"}}
  • Float value format : {{float_value|floatformat}}
  • Float value with decimal value limit : {{float_value|floatformat:3}}
  • Float value -3 (-3 means display only 3 positive decimal values other don't display(0 decimal not displayed))) : {{float_value1|floatformat:"-3"}}
  • if condition : {%if num == 1 and name == 'narayana'|title %}1{% endif%}
    • Hint: you can use different operators in DTL(Django Template Language) +, -, ==, !=, >, <, >=, <==, and, or, not, is, is not
  • if with length condition example : {% if num|floatformat:"0"|length >= 1 %}Grater Value{%else%}Lowest value{% endif %}
  • Dot Lookup example: {{test}}
  • Dot Lookup example: {{test.0}}
  • Dot Lookup example: {{test.0.bar}}
  • Dot Lookup example: {{test.0.bar.0}}
  • For loop Example :

  • Reverse list in for loop using reversed :

      {% for i in list2 %}

Predefind for loop variables

  • forloop.counter :
    {%for m in list1 %} {{forloop.counter}}{{m}}
    {% endfor %}
  • For loop counter0 :
    {% for m in list1 %} {{forloop.counter0}}{{m}}
    {%endfor%}
  • forloop using revcounter :
    {% for n in list1 %} {{forloop.revcounter}} {{n}}
    {%endfor%}
  • forloop using revcounter from 0 :
    {% for n in list1 %} {{forloop.revcounter0}} {{n}}
    {%endfor%}
  • forloop check first element :
    {%for i in list1%} {{forloop.first}}{{i}}
    {%endfor%}
  • forloop using last element check :
    {%for f in list1 reversed%}{{forloop.last}}{{f}}
    {%endfor%}
  • print numbers for Parent for loop in nested loop : {%for key,value in data1.items%}
    {{key}}
    {%for k,v in value.items%} {{forloop.parentloop.counter}} {{k}}{{v}}
    {%endfor%} {%endfor%}
